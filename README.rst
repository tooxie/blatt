Blatt
=====

Newspapers of the world, unite!


------------
Installation
------------

* Clone the project::

.. code:: sh

    $ git clone https://github.com/tooxie/blatt.git
    $ cd blatt

* Create a virtualenv::

.. code:: sh

    $ virtualenv .venv
    $ source .venv/bin/activate

* Install dependencies::

.. code:: sh

    $ pip install -r requirements.pip
